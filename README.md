# QuickSwitch

A lightweight Windows utility that remaps your CapsLock key to switch keyboard languages.

## What it does

- **CapsLock** → Switch keyboard language (Shift+Alt)
- **Shift+CapsLock** → Toggle CapsLock on/off

## Requirements

- Windows 10/11

## Installation

1. Download the latest release
2. Run `QuickSwitch.exe`

## Building from source

```bash
git clone https://github.com/AAlshahrani1/QuickSwitch
cd QuickSwitch
dotnet build
```
